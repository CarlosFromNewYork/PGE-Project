How to use the custom graphics

In some times need to replace the global images by your own. You can do it without overwriting of the images. You can simulate many of the items you will need to make your game episode unique and colorful on your own.

The process is quite simple, but for those of you who don't know how, here is an in depth look at using custom graphics.

Customizing notes

- On customizing of animated items be careful, your sprite must be have the same frames number which in the original image.

- If you creating the custom BGO's, please, be careful by using these graphics in the SMBX. If size of custom BGO larger than original item image which you replacing, it will be cropped in the SMBX. Don't use larger images if you wanna use levels in the SMBX. Same for a Tiles, Sceneries, Paths and Levels on the world map.

- On customizing NPC's, you must create the image sprite with same image size and frames number. If you create custom image size, you should to create the npc-*.txt config file for define the properties for your custom NPC.

 

 

 

Graphics formats support

 

The PGE support these graphics formats:

Format Description Notes
GIF CompuServe GIF, witout transparency (with using of the masks)
GIF CompuServe GIF, with own transparency SMBX will fill transparency by black
BMP Uncompressed image format (with using of the masks)  
PNG Portable Network Graphics, with alpha-channel transparency Not support by SMBX
JPG  Joint Photographic EG, Lossy-data compressed Not support by SMBX
TIF Tagged Image File Format Not support by SMBX
TGA Truevision TGA Not support by SMBX
