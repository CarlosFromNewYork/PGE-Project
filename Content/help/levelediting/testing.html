How to run test with SMBX 1.3

Before engine part will made, you can run tests of levels inside internal SMBX editor.

You need to use only SMBX 1.3 or 1.3.0.1, saved by PGE Editor files are not compatible with old versions of SMBX.

 

1) Run PGE Editor first and open necessary level or create and save them.

2) Open the SMBX in editor mode. Both application must be runned!  (PGE Editor and SMBX)

3) Open by SMBX Editor same level file which already opened by PGE Editor and leave them, minimize the SMBX window.

 

Click on the image for see big image:

 

4) Make the necessary modifications by PGE Editor and save file.

 

 

5) On question about saving file, press the "No" if you want run test with edited by PGE file:

 

And test of edited file was runned!
